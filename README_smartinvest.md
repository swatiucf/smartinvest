Project :SmartInvest - AI-Powered Personal Finance & Robo-Advisory Platform
Overview
SmartInvest is a full-stack web application that leverages artificial intelligence to democratize access to professional-grade financial insights. The platform helps users make informed investment decisions through automated analysis, personalized recommendations, and real-time market data visualization.

Problem Statement
Modern consumers face overwhelming investment choices but often lack the financial literacy or time to analyze markets effectively. SmartInvest bridges this gap by providing AI-driven tools that simplify financial decision-making
Secure Authentication - Firebase-based user authentication
•	 Financial Profiling - Comprehensive user profile with income, goals, and risk tolerance
•	 Real-Time Market Data - Live stock prices via Alpha Vantage API
•	 Cryptocurrency Tracking - Real-time crypto data via CoinGecko API
•	 AI Recommendations - Personalized investment suggestions powered by OpenAI
•	 Interactive Dashboard - Visual representation of portfolio and market trends

Technology Stack
Frontend
•	React.js - Component-based UI framework
•	Tailwind CSS - Utility-first CSS framework
•	Vite - Next-generation frontend tooling
•	Firebase SDK - Authentication and database

Backend
•	Node.js - JavaScript runtime
•	Express.js - Web application framework
•	REST API - RESTful architecture
Database & Authentication
•	Firebase Authentication - Secure user management
•	Cloud Firestore - NoSQL database for user profiles
